## Features

### Markdown-Native Coordination

Manage work through simple task files instead of hidden agent state or fragile chat context.

### Deterministic Task Ownership

Assign each task to a specific agent with clear responsibilities, dependencies, and completion notes.

### Workflow That Stays Visible

Track work across inbox, ready, in-progress, review, and done using a board structure that remains human-readable at every step.

### Built for Specialized Agents

Let orchestrators, builders, reviewers, and content agents focus on narrow roles without overlapping work.
