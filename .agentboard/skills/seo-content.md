# SEO Content Skill

## Purpose

Use this skill when writing documentation, README content, or public-facing explanatory content that should be clear, structured, and easy to discover.

## Applies To

- content-creator
- reviewer when validating documentation

## Guidance

- Put definitions before procedures.
- Use headings that match user questions.
- Explain constraints and intentionally omitted features directly.
- Link from high-level docs to deeper docs when a topic grows.
- Keep examples short and concrete.

## Out Of Scope

- Marketing landing pages.
- Keyword stuffing.
- External analytics or SEO tooling.
